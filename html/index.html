Welcome, to the Xbox orignal Api

How does this work, you may ask

To use the Api you need to go to /games

If you use /games/4 it will get the 4th game on the list.

You can find the game just by ID, to do this /games/4 it will get the 4th game on the list.

You can do the same for games genres, /games/genre/sports it will show all the sports games
