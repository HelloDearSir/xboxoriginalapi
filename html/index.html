Xbox Original API

Simple REST API for Xbox game and genre data

Overview

The Xbox Original API provides programmatic access to Xbox game data, including titles and genres. Perfect for dashboards, hobby projects, and integrations.

Base URL

https://xboxoriginalapi-production.up.railway.app

Endpoints

GET /games

List all games

GET /games/{id}

Get a specific game by ID

GET /games/genre/{genre}

Get games by genre

Example Request 

curl https://xboxoriginalapi-production.up.railway.app/games/1

Example Response 

{
"id":1,
"title":"187 Ride of Die",
"Developers":"Ubisoft Paris",
"Publishers":"Ubisoft",
"Pal":"Aug 25 2025",
"JP":"Unrelased",
"NA":"Aug 23 2005",
"genres":["arcade racing"]
}

JavaScript Example 

fetch('https://xboxoriginalapi-production.up.railway.app/games/1')
  .then(res => res.json())
  .then(data => console.log(data));
